## what blocks you

peer review **r8: behavior-intent-coverage** flags one blocker that is a
verifiable false positive. the reviewer claims the `declastruct-github` pin was
not raised from `minVersion('1.5.5')` to `minVersion('1.6.0')` in
`src/practices/provision-github/best-practice/package.json`.

the pin bump **is** present. `git diff main` proves it:

```
diff --git a/src/practices/provision-github/best-practice/package.json
-    "declastruct-github": "@declapract{check.minVersion('1.5.5')}"
+    "declastruct-github": "@declapract{check.minVersion('1.6.0')}"
```

root cause: the automated reviewer's scope is `**/*.{ts,sh}` intersected with
the diff-since-main. that scope **structurally excludes `.json` files**, so the
reviewer cannot see the `package.json` line where the pin lives. it therefore
concludes the requirement is unmet, when in fact it is met.
