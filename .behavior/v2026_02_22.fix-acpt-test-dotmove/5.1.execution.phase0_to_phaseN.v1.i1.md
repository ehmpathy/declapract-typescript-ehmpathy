### phase 1: bisect root cause

**status**: complete (skipped detailed bisect — root cause clear from analysis)

root cause: `**/*.ts.declapract.ts` pattern only matches `.ts` files. files like `.json` and `.yml` in hidden directories are never considered by extension-specific patterns.

### phase 2: implement fix

**status**: complete

#### 2.1 create catch-all declapract file

created: `src/practices/tests/bad-practices/old-acceptance-dir-location/accept.blackbox/**/*.declapract.ts`

the catch-all pattern matches ANY file type, not just `.ts`:
```ts
export const check = FileCheckType.EXISTS;
export const fix: FileFixFunction = (contents, context) => {
  const newPath = context.relativeFilePath.replace(/^accept\.blackbox\//, 'blackbox/');
  return { contents: contents ?? null, relativeFilePath: newPath };
};
```
