**rule**: .agent/repo=ehmpathy/role=mechanic/briefs/practices/code.prod/pitofsuccess.typedefs/rule.forbid.as-cast.md.min

**locations**:
- src/practices/runtime-type-checking/bad-practices/joi-imports/src/**/*.ts.declapract.test.ts:9

- src/practices/runtime-type-checking/bad-practices/simple-lambda-handlers/src/**/*.ts.declapract.test.ts:9

Test files use `{} as any` to mock the context argument in multiple test cases. This is an undocumented `as` cast, which is forbidden by the rule unless it is at an external boundary with a documented reason.
